Telegram-бот для генерации изображений через AI.
Оплата — Telegram Stars.

================================================================

?? ФУНКЦИИ

  • Генерация картинок по текстовому описанию
  • Оплата звёздами Telegram
  • Тарифы: 1, 10, 100, 1000 генераций
  • Реферальная система (10% от трат реферала)
  • Профиль с балансом и статистикой
  • Поддержка (сообщения админу с возможностью ответа)
  • Уникализация промптов (каждая картинка разная)
  • Администраторы с бесконечными генерациями

================================================================

?? ТАРИФЫ

  1 генерация   — 1 звезда
  10 генераций  — 10 звезд
  100 генераций — 50 звезд
  1000 генераций — 500 звезд

================================================================

?? ФАЙЛЫ ДЛЯ ДЕПЛОЯ

  BotImage.py        — основной код
  requirements.txt   — зависимости
  Procfile           — для Render/Heroku
  runtime.txt        — версия Python

================================================================

?? ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ

  BOT_TOKEN=токен_бота
  ADMIN_IDS=123,456  (опционально)

  pip install -r requirements.txt

  /help     — справка
